# trunc8's blog

### Steps to replicate
```sh
apt install gem
export GEM_HOME="$HOME/gems"
export PATH="$HOME/gems/bin:$PATH"
gem install jekyll bundler
```


### For rouge themes
```sh
gem install kramdown rouge
rougify style monokai.sublime > assets/css/syntax.css
```
